# 🎲 Simple Fancy Monopoly Game
A text-based Monopoly game for 2 players, with houses, chance cards, and buyable properties.  
No extra installs — just run with Python!

## 📋 Features
- 2-player mode (perfect for quick games)
- Buyable properties (with prices and rent)
- House system (buy houses to increase rent!)
- Chance cards (random fun effects: gain/lose money, move, free houses)
- Bankruptcy win/lose condition
- Pass Go = collect $200

## 🚀 How to Play
1. Clone this repo or download the files
2. Run the game:
   ```bash
   python monopoly.py
   ```
3. Follow the on-screen instructions:
   - Press Enter to roll the dice
   - Buy properties (if you have enough money)
   - Buy houses to make more rent
   - Avoid going bankrupt!

## 🎯 Game Rules (Simple Version)
- Start with $1500 each
- Roll dice to move around the board
- Buy properties to collect rent from the other player
- Buy houses to increase rent (costs $50 per house)
- Draw chance cards for random surprises
- First player to go bankrupt loses
